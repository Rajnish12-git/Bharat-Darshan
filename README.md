**Bharat Darshan** is a modern web platform that showcases the rich cultural heritage of India. Users can explore Indian states, monuments, festivals, cuisines, art forms, and traditions through an interactive and visually engaging experience.

## 🌐 Live Demo

🔗 https://web-bharat-darshan.vercel.app/

## 📂 GitHub Repository

🔗 https://github.com/Rajnish12-git/Bharat-Darshan

## ✨ Features

* Explore Indian states and their cultural heritage
* Discover famous monuments and historical landmarks
* Learn about regional cuisines and traditional dishes
* Explore festivals, traditions, and art forms
* Secure user authentication using Firebase Authentication
* Dynamic content management with Cloud Firestore
* Responsive design for desktop and mobile devices
* Fast and optimized user experience with Next.js

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Rajnish12-git/Bharat-Darshan.git
cd Bharat-Darshan
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file and add your Firebase configuration:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## 🎯 Project Objective

The objective of Bharat Darshan is to digitally preserve and present the diverse culture of India through an interactive platform that is educational, informative, and accessible to everyone.
